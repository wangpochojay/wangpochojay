About My Project

This personal portfolio website project is created as a hands-on practice to improve my understanding of HTML and CSS outside the classroom. The main goal of this project is to strengthen my basic web development skills by building a real, structured, and responsive website using only HTML and CSS.

Through this project, I am learning how to design and organize web pages using Flexbox, Grid, and Media Queries. It also helps me understand how different sections of a website such as Home, About, Portfolio, Hobbies, and Contact are connected and styled in a consistent way.

Instead of only studying theory, this project allows me to apply what I learn in a practical way. It improves my creativity, problem-solving skills, and confidence in building websites from scratch.

Overall, this project is an important step in my journey to become a better web designer and developer.
